# Howrah Railway Institute – React Clone

This is a React single-page clone of the [`howrahrailwayinstitute.in`](https://howrahrailwayinstitute.in/) marketing site, built with a clean component structure.

## Tech Stack

- React 18
- Vite
- TypeScript (for main/app/components)

## Structure

- `src/App.tsx` – top-level layout
- `src/components/Header.tsx` – sticky navigation
- `src/components/HeroBooking.tsx` – hero text and booking form
- `src/components/InstitutesSection.tsx` & `InstituteCard.tsx` – institutes grid
- `src/components/Footer.tsx` – footer with links and simple form
- `src/styles.css` – global layout & visual styles

## Getting Started

```bash
npm install
npm run dev
```

Then open the printed local URL (usually `http://localhost:5173`) in your browser.
